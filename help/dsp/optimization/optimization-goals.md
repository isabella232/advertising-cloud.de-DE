# Optimierungsziele und Verwendung

| Optimierungsziel | Beschreibung | Verwendungszweck |
| -----------| ---------- | ---------- |
| [!UICONTROL Always Max Bid & Highest Clickthrough Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuweisung Platzierungen mit den höchsten Clickthrough-Raten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird der CTR Vorrang eingeräumt, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer der Satz [!UICONTROL Max Bid], aber der prognostizierte CTR-Schwellenwert wird strenger, wenn eine Platzierung gut ausgegeben wird. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Clickthrough-Rate<br><br>Anzeigentypen: Pre-roll, Display<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie ein festes CPM-Ziel haben, das nicht überschritten werden muss, und ein CTR-Ziel, das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel, und DSP wird beim Versuch, das gesamte Budget auszugeben, die bestmögliche CTR erreichen. |
| [!UICONTROL Always Max Bid & Highest Completion Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuweisung Platzierungen mit den höchsten Abschlussraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Abschlussrate priorisiert, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer das festgelegte Max-Angebot, aber der prognostizierte Schwellenwert für die Abschlussrate wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Abschlussrate<br><br>Anzeigentypen: Nur Pre-Roll<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie über ein festes CPM-Ziel verfügen, das nicht überschritten werden muss, und ein Ziel mit der Abschlussrate , das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel. DSP wird beim Versuch, das gesamte Budget auszugeben, die bestmögliche Abschlussrate erzielen. |
| [!UICONTROL Always Max Bid & Highest Engagement Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuweisung Platzierungen mit den höchsten Interaktionsraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Interaktionsrate priorisiert, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer das festgelegte Max. Angebot, aber der prognostizierte Schwellenwert für die Interaktionsrate wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Interaktionsrate<br><br>Anzeigentypen: Nur für mobile Zwischenräume<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie über ein festes CPM-Ziel verfügen, das nicht überschritten werden muss, und über ein Interaktionsziel verfügen, das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel. DSP erzielt beim Versuch, das gesamte Budget auszugeben, die bestmögliche Interaktionsrate. |
| [!UICONTROL Always Max Bid & Highest Viewability Rate (Adobe – GroupM)] | Bei der Optimierung auf Paketebene wird bei der Budgetzuordnung Platzierungen mit den höchsten Sichtbarkeitsraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Sichtbarkeitsrate priorisiert, wenn die Ausgabenziele erreicht werden. Angebot auf Platzierungsebene.<br><br>Wenn die Einstellung Sichtbarkeitsempfindlichkeit der Kampagne auf &quot;Standard (50 % der Anzeige für 2 aufeinander folgende Sekunden)&quot;festgelegt ist, wird der Standard für die Sichtbarkeitsmessung des Media Rating Council (MRC) für die Kampagne verwendet. Wenn die Kampagne auf &quot;Streng (100 % der Anzeige in Ansicht und Audio für 50 % Dauer)&quot;festgelegt ist, wird der GroupM-Standard für die Sichtbarkeitsmessung für die Kampagne verwendet. Idealerweise sollten Sie die Kampagneneinstellung mit dem Optimierungsziel und den Filtereinstellungen vor dem Angebot abgleichen. |
